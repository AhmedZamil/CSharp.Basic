In a card game Each player will be given a set of random cards. Players will throw
on the table their own WINNING CARD .The player wiht the heighest Card wins

A winning Card is the card that only exist once in the set of cards and heigst one.

Given an array of sets of integers cards return the card of the winning player. 
Return -1 if no such card found.


Example 1

Input : cards =[[5,7,3,9,4,9,8,3,1],[1,2,2,4,4,1],[1,2,3]] 
output : 8

Example 2
Input : cards = [[5,5],[2,2]]   // 5 5, 2 2
output : -1
